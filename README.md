# Курсы валют — десктоп-приложение с GUI

Простое приложение на Python, которое получает актуальные курсы валют
с открытого публичного API Центрального Банка РФ, позволяет
фильтровать и сортировать данные через графический интерфейс,
а затем сохранять результат в JSON-файл.

## Возможности

- Получение актуальных курсов валют без регистрации и API-ключа.
- **Автоматическая загрузка курсов сразу при запуске** — данные
  появляются в таблице, как только приложение откроется, нажимать
  кнопку «Обновить курсы» не обязательно.
- Графический интерфейс на `tkinter` (входит в стандартную поставку
  Python, дополнительно ничего ставить не нужно), полностью на
  русском языке.
- **Мгновенное применение фильтров** — любое изменение текста, диапазона
  курса, поля сортировки или направления сортировки сразу же
  обновляет таблицу, без отдельной кнопки «Применить».
- Фильтрация по коду/названию валюты **регистронезависима**: можно
  вводить запрос в любом регистре — `usd`, `USD`, `UsD`, `евро`,
  `ЕВРО` дадут одинаковый результат.
- Сортировка по коду, названию, значению курса, абсолютному и
  процентному изменению — по возрастанию или убыванию.
- Сохранение отфильтрованного результата в JSON-файл через
  стандартный диалог сохранения.
- Логирование событий и понятные сообщения об ошибках сети.

## Источник данных

Используется бесплатное зеркало официального API ЦБ РФ:
`https://www.cbr-xml-daily.ru/daily_json.js`. Регистрация, API-ключ
и авторизация не требуются — это полностью открытый источник.

## Установка

Требуется Python 3.9 или новее.

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

После запуска:

1. Приложение **сразу же автоматически** загружает актуальные курсы
   с сервера ЦБ РФ и показывает их в таблице — отдельных действий
   не требуется. Кнопка **«Обновить курсы»** позволяет загрузить
   данные заново в любой момент.
2. Любое изменение фильтров — текст в поле «Код или название
   валюты», диапазон «Курс от / Курс до», выбор поля сортировки или
   флажок «По убыванию» — **применяется мгновенно**, таблица
   обновляется автоматически.
3. Нажмите **«Сохранить в JSON»** и выберите место сохранения файла,
   чтобы сохранить текущий отфильтрованный и отсортированный список.

### Поля фильтров

| Поле                          | Описание                                                                 |
|--------------------------------|---------------------------------------------------------------------------|
| «Код или название валюты»     | Поиск подстроки по буквенному коду (`USD`) или названию («Доллар США»). Регистр не важен. |
| «Курс от» / «Курс до»          | Диапазон значения курса к рублю (включительно). Поддерживает точку и запятую как разделитель. |
| «Сортировать по»               | Поле сортировки: «Код валюты», «Название», «Курс», «Изменение», «Изменение, %». |
| «По убыванию»                  | Переключает направление сортировки между возрастанием и убыванием.        |

## Формат сохраняемого JSON

```json
{
  "source_date": "2026-06-30T11:30:00+03:00",
  "saved_at": "2026-06-30T14:22:10",
  "count": 2,
  "rates": [
    {
      "char_code": "USD",
      "name": "Доллар США",
      "nominal": 1,
      "value": 90.1234,
      "previous": 89.5,
      "change": 0.6234,
      "change_percent": 0.6964
    },
    {
      "char_code": "EUR",
      "name": "Евро",
      "nominal": 1,
      "value": 98.0,
      "previous": 99.0,
      "change": -1.0,
      "change_percent": -1.0101
    }
  ]
}
```

Поля:

| Поле             | Описание                                          |
|------------------|----------------------------------------------------|
| `source_date`    | дата актуальности курсов по данным ЦБ РФ           |
| `saved_at`       | время сохранения файла на локальной машине          |
| `count`          | количество записей в файле                          |
| `char_code`      | буквенный код валюты (ISO 4217)                     |
| `name`           | название валюты на русском языке                    |
| `nominal`        | номинал (за сколько единиц валюты указан курс)       |
| `value`          | текущий курс к рублю                                 |
| `previous`       | предыдущее значение курса                            |
| `change`         | абсолютное изменение курса                           |
| `change_percent` | изменение курса в процентах                          |

## Известные ограничения

- Приложению требуется доступ в интернет для получения курсов.
- Используется один источник данных (ЦБ РФ); для использования
  альтернативного API (например, exchangerate.host) потребуется
  изменить функцию `fetch_rates` в `api_client.py`.
- Графический интерфейс рассчитан на десктоп (Windows/macOS/Linux)
  с установленным `tkinter`. На некоторых Linux-дистрибутивах его
  нужно установить отдельно, например: `sudo apt install python3-tk`.
- Мгновенное применение текстового фильтра означает, что фильтрация
  запускается при каждом нажатии клавиши в поле ввода; так как
  список валют (несколько десятков записей) обрабатывается локально
  и без сетевых запросов, заметных задержек это не вызывает.

## Чего в проекте намеренно нет

В проекте отсутствуют функции автоматической регистрации аккаунтов,
ротации/смены IP-адреса и сохранения cookie-файлов для обхода защит
сторонних сайтов. Используемый источник данных открытый и не требует
авторизации, поэтому такие механизмы не нужны и не реализовывались.
